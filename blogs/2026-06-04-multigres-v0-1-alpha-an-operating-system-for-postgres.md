Multigres v0.1 alpha introduces a scalable operating system for Postgres that delivers horizontal scaling, high availability, and operational simplicity through features like advanced connection pooling and automatic failovers. The release includes a Kubernetes operator for deployment and a consensus-based approach to managing replication and preventing data loss. While the alpha is ready for experimentation and feedback, sharding capabilities remain forthcoming and the software is not yet suitable for production environments.
