OpenCode, an open-source AI coding agent, now integrates with Supabase, giving agents access to Supabase account and project management APIs plus bundled Supabase skills. The integration lets agents work with databases, Edge Functions, and logs directly, so they can verify their work and deploy functions without leaving the coding environment.
